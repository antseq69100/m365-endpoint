1. Download and extract Greenshot-PORTABLE-1.3.315-RELEASE.zip here : https://getgreenshot.org/version-history/
2. Copy the extracted Greenshot files into .\package-layout\Files\Greenshot\
3. Copy PSAppDeployToolkit into .\package-layout\PSAppDeployToolkit\
4. Copy Invoke-AppDeployToolkit.ps1 to the package root
5. Build the Intune package with IntuneWinAppUtil.exe

Example:
IntuneWinAppUtil.exe -c .\package-layout -s Invoke-AppDeployToolkit.ps1 -o .\output
